Описание:

Время закупки для AWP.
Запрещает покупать AWP через N сек. после старта раунда.

Решает проблему когда некоторые игроки специально уходят в спеки при 4 на 5 и ждут
пока команда покинет респу потом заходят за команду и закупают авп.

Требования:

Metamod

Sourcemod

Для компиляции пожеланию зависимо от игры:

More Colors

[INC] CS:GO Colors

Ядро плагина - [ClientMod] API

Для работы на CSSv34 отредактируйте .sp файл поменяв на строке 3
#define CSSv34 false на true

Конфиг создаться автоматически в cstrike/cfg/sourcemod/[OS]BuytimeAWP.cfg
